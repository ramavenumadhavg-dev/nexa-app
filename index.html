Anthropic API key (fallback)
Checking…
Stored only on this device. Needed when you run Nexa outside the Claude preview. Get one at console.anthropic.com. By default it's saved in plain browser storage — tap "Lock this key" to encrypt it (AES-256) so it can't be read from storage without your passphrase, which is never itself stored anywhere.
Image generation key
Optional — unlocks ChatGPT-grade images
Routes image generation to gpt-image-2 (the model ChatGPT uses), plus Nano Banana & Seedream — far sharper and structurally correct vs. the free Flux fallback. Free key at enter.pollinations.ai.
Google (Drive, Docs, Gmail)
Not connected
Lets Nexa import a file from Drive, read a Google Doc/Sheet, or search Gmail from the attach menu. Needs your own free OAuth Client ID from console.cloud.google.com (add this page's URL as an authorized JavaScript origin). Stored only on this device; nothing is sent anywhere but Google.
Accounts & sync
Add your Supabase project to enable accounts
Free at supabase.com — create a project, then paste its URL + anon key here. Run the one-time SQL in the code comment above initSupa() to create the sync table. Enables real email/password accounts and syncs chats, notes, projects & prefs across every device you sign into.
Search plugin (Tavily)
Optional extra search source for tools/plugins
Nexa already searches the live web via Claude's built-in search tool on every message. Adding a free Tavily key (1,000 searches/month, no card, at tavily.com) plugs in a second, independent search + citation source Nexa can call as a tool — and is the pattern to follow for adding any other tool/plugin.
